Getting Started

Essential information for new users of the OneVictoria platform

OneVictoria is the official resident engagement platform for the Municipality of Victoria. This digital hub connects citizens with municipal services, community events, volunteer programs, and civic announcements. The platform enables residents to:

  • Register for community events and programs
  • Submit volunteer applications for municipal initiatives
  • Track community service hour contributions
  • Make monetary or in-kind donations to support local causes
  • Receive real-time notifications regarding community updates
  • Manage personal profile information securely

Account registration is straightforward:

  1. Navigate to the OneVictoria portal homepage
  2. On the authentication panel, select "Create Account"
  3. Complete the registration form with required fields (full name, email, phone, age, gender, address)
  4. Create a secure password (minimum 6 characters)
  5. Click "Create Account" to finalize

Recommendation: Use an active email address for account recovery and notifications.

  1. Visit the OneVictoria portal
  2. Enter your registered email and password
  3. Select "Access Portal"

The dashboard uses a sidebar navigation with sections for Announcements, Events, My Events, Volunteer, Service Hours, and Donations.

Account Management

Managing your profile, credentials, and account settings

  1. Go to Profile → Edit Profile
  2. Update mobile, address, or profile picture
  3. Save changes

Name, email, gender, and age are immutable; contact support for changes.

Profile → click current image → select file (JPG/PNG) → save.

Profile → Edit Profile → enter new password → save.

Events & Registration

Discovering and managing community events

Events tab → click Register on desired event → confirmation appears.

My Events → click Cancel on the event card. Please cancel 24h before.

Volunteering

Volunteer tab → select skill, upload verification, set availability → Submit.

Service Hours

Hours are earned by attending registered events and being certified by organizers.

Donations

Donations tab → fill form → choose payment (GCash, Maya, bank, cash) → confirm.

Notifications

Click bell icon in sidebar or navigate to Notifications tab.

Troubleshooting

Contact support with your registered email for a reset procedure.
